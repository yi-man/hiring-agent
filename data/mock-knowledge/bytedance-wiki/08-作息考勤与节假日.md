# 【模拟 Wiki】作息、考勤与节假日

> 模拟员工手册摘要。真实政策以劳动合同、员工手册与属地法规为准。

## 标准工作时间（模拟）

- **常规**：工作日每天约 8 小时工时口径（午休另计，视园区）。
- **常见班次示例**：10:00–19:00（含午休）或 9:30–18:30 等；**以团队与城市政策为准**。
- **弹性**：部分研发团队允许在核心协作时间（例如 11:00–16:00）在岗，其余弹性（演示政策）。

## 考勤（模拟）

| 项目 | 说明                                        |
| ---- | ------------------------------------------- |
| 打卡 | 门禁 / 飞书考勤（视城市）                   |
| 迟到 | 有宽限与累计规则，频繁迟到影响绩效印象      |
| 请假 | 年假、病假、事假、婚假、产假/陪产假、丧假等 |
| 出差 | 出差期间按差旅规则，考勤记为公出            |
| 补卡 | 忘记打卡可按规定次数申请                    |

## 加班与调休（模拟原则）

1. 鼓励提升效率，不提倡无效加班。
2. 因项目需要的加班，应提前与上级对齐预期。
3. 法定节假日加班依法支付或安排补休（以属地法律与公司制度为准）。
4. 部分岗位综合工时或不定时工时（少见，以合同约定）。

## 法定节假日

按照中国大陆法定节假日安排休息（元旦、春节、清明、劳动节、端午、中秋、国庆等），具体调休以国务院与公司通知为准。

## 公司额外福利假（模拟，可能因年而异）

| 类型                | 演示口径                                     |
| ------------------- | -------------------------------------------- |
| 带薪年假            | 按司龄阶梯，例如满 1 年 X 天起（具体看制度） |
| 全薪病假            | 每年有额度，超额按制度折算                   |
| 生日假 / 周年假     | 部分年份有，需提前申请                       |
| 志愿者假 / 家长会假 | 偶发政策，以当年通知为准                     |
| 春节延长            | 偶有「公司假」补充，不是每年保证             |

## 产假 / 陪产假 / 育儿相关（模拟提示）

- 按国家与地方规定执行，公司可能有补充福利。
- 具体天数、津贴以 HR 政策与社保规则为准。
- 哺乳期工时与母婴室资源在部分园区提供。

## 作息相关候选人 FAQ

**Q：是 996 吗？**  
A：（模拟口径）不同团队节奏不同。内容与增长团队在大版本或活动期更忙；飞书等 ToB 团队更贴近客户交付节奏。面试时请直接问「最近一个季度团队平均下班时间/值班制度」。

**Q：周末要值班吗？**  
A：在线业务可能有 oncall。有排班、补偿或调休机制（视团队）。

**Q：能晚到早走吗？**  
A：在弹性政策与核心时间满足的前提下，部分团队可以；需与 TL 对齐。
