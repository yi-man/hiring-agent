# 【模拟 Wiki】薪酬福利与待遇

> 模拟 Total Rewards 手册摘要。数字用区间与结构说明，避免伪造精确薪级表。

## 薪酬结构（模拟）

典型正式员工包由以下部分组成（岗位不同权重不同）：

1. **月薪（基本工资 + 岗位工资等）**
2. **年终奖 / 绩效奖金**（与个人与团队绩效相关）
3. **长期激励（如期权/RSU 类，演示名「股票激励」）**——以 Offer 是否授予为准
4. **津贴补贴**（餐补、交通、通讯、高温等，城市差异大）
5. **法定福利**：五险一金（及地方补充）

### 演示公式

> 年包 ≈ 月薪 × 12 + 年终奖目标 +（长期激励当年估值，若有）+ 现金补贴

## 薪资定级逻辑（模拟）

- 看：专业级别、城市、稀缺度、面试表现、竞品 offer。
- 校招与社招带宽不同。
- 调薪通常与晋升/年度评审相关，不是每月调整。

## 奖金（模拟）

| 类型     | 说明                                  |
| -------- | ------------------------------------- |
| 年终奖   | 常见 0–N 个月月薪口径，与绩效系数相关 |
| 项目奖   | 重大项目临时激励（不保证）            |
| 销售提成 | 销售/部分商业化岗位特有               |
| 即时认可 | 小额奖励、感谢金（文化项目）          |

## 长期激励（模拟说明）

- 授予、归属（vesting）周期、行权/兑现规则以协议为准。
- 候选人应关注：是否授予、归属年限、离职条款、税务。
- Agent 不得承诺「一定上市」或具体估值。

## 五险一金与补充福利（模拟）

- **养老保险、医疗保险、失业保险、工伤、生育** + **住房公积金**
- 缴纳基数与比例按城市政策与公司方案
- **补充医疗保险 / 商业保险**（演示：含门诊、住院、子女险等可选包）
- **年度体检**
- **员工心理咨询热线**（EAP，若有）

## 日常补贴（模拟菜单）

| 项目        | 常见形态                                       |
| ----------- | ---------------------------------------------- |
| 餐补        | 食堂刷卡额度或补贴                             |
| 交通        | 班车 / 交通补贴                                |
| 住房        | 租房补贴或人才公寓（城市与职级相关，不人人有） |
| 通讯        | 少数岗位                                       |
| 节日礼品    | 节假日礼盒或积分                               |
| 图书 / 学习 | 培训预算、内部分享、在线课程                   |

## 假期与休假变现

- 年假优先安排休完；应休未休的处理按制度（补偿或结转规则因年而异）。
- 病假、事假扣薪规则在员工手册。

## 离职与补偿（原则性）

- 试用期与转正后解除/终止按劳动法与合同。
- 竞业限制若适用，有补偿金条款。
- 长期激励未归属部分通常按协议失效。

## 候选人谈判建议（模拟）

1. 先明确级别与城市带宽。
2. 比较年包结构，而不是只比月薪。
3. 问清年终奖目标、是否有股票、几险几金基数。
4. 把「学习成长、业务阶段、团队口碑」一并纳入决策。

## Agent 话术红线

- 不承诺具体 Offer 金额。
- 可介绍结构与福利种类。
- 精确数字引导至 HR / 面试官。
