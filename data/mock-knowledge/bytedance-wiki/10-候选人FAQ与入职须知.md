# 【模拟 Wiki】候选人 FAQ 与入职须知

> 模拟校招/社招知识库。用于招聘助手回答候选人高频问题。

## 招聘流程（模拟）

1. 投递 / 内推 / 猎头推荐
2. 简历评估
3. 业务面试（1–多轮，含编码/系统设计/产品案例等）
4. 交叉面或 HR 面
5. 材料与背景调查（视岗位）
6. Offer 沟通与发放
7. 接受 Offer → 入职准备 → 报到

### 时效

- 面试反馈常见数个工作日到两周不等。
- 大版本招聘季可能更长。
- 候选人可主动礼貌催问进度。

## 面试准备建议

### 研发

- 算法与编码题、系统与项目深挖、线上问题排查思路
- 了解所面业务（抖音 / 汽水 / 飞书等）的用户场景
- 准备「Owner 故事」：你如何推动结果

### 产品

- 产品案例、指标拆解、取舍与复盘
- 对竞品与用户洞察有自己的观点

### 设计

- 作品集叙事、设计系统与业务目标对齐

### 运营 / 市场 / 销售

- 增长实验、活动复盘、客户案例、数据分析能力

## Offer 与谈薪

- Offer 一般含：职位、级别、城市、薪资结构、入职日期、主体公司。
- 有效期通常有限，超时可能失效。
- 对比多家时，关注年包、岗位内容、团队、成长。

## 入职材料（模拟清单）

- 身份证、学历学位、离职证明（社招）
- 照片、银行卡、户口/居住信息（社保用）
- 体检报告（按通知）
- 竞业或保密相关声明（如有）

## 试用期（模拟）

- 常见 3–6 个月（合同约定）。
- 试用期有目标与中期反馈。
- 通过后转正；未达预期可能解除（依法依规）。

## 内推

- 员工内推通常有奖励政策（金额与规则变化）。
- 内推不保证面试通过，但可能加快简历触达。

## 候选人最关心的 20 问（模拟标准答）

1. **公司是做什么的？** → 见《公司概览》《业务矩阵》。
2. **抖音和头条什么关系？** → 同生态不同产品形态。
3. **汽水音乐前景？** → 成长投入期，看增长与版权经营。
4. **飞书是 ToB 吗？** → 是，偏协作与企业服务。
5. **加班多吗？** → 团队差异大，建议面询。
6. **有年假吗？** → 有，按司龄与制度。
7. **有股票吗？** → 看 Offer 是否授予。
8. **在哪个城市？** → 以招聘需求城市为准。
9. **能 Remote 吗？** → 默认办公室，特殊审批。
10. **培训体系？** → 新人训、导师制、内部分享、在线课程（演示）。
11. **晋升周期？** → 常见按评审周期，看贡献。
12. **女生友好吗？** → 有产假等法定与补充政策，团队氛围因组而异。
13. **外地户籍？** → 关注积分落户与租房补贴政策（城市差异）。
14. **英文要求？** → 国际化/跨区协作岗位更高。
15. **技术栈？** → 因团队而异：Go/Python/C++/Java/RN 等均可能。
16. **试用期工资？** → 依法不得低于约定与当地最低标准，以合同为准。
17. **五险一金基数？** → 按政策与公司方案。
18. **竞业吗？** → 部分岗位可能涉及，看协议。
19. **简历多久有结果？** → 建议 1–2 周内跟进。
20. **入职带什么？** → 证件材料 + 好学心态。

## 入职第一周（模拟 Onboarding）

1. 领用工卡与电脑，开通飞书与代码权限
2. 完成安全、合规、隐私培训
3. 与 Mentor / TL 对齐 30-60-90 天目标
4. 阅读团队 Wiki 与值班手册
5. 提交第一个小改动 / 小需求打通流程

## 行为准则摘要（模拟）

- 保护用户数据与公司机密
- 禁止贿赂与利益冲突
- 尊重同事，拒绝骚扰歧视
- 对外发言走公关与合规流程
- 使用正版与合规开源许可

## 应急联系（演示占位）

- 招聘进度：招聘 HR / 业务接口人
- 入职当天：行政 / Onboarding 群
- 系统权限：IT 服务台（飞书工单）
