# 【模拟 Wiki】使用说明

> **重要声明**：本知识库全部内容为 **Hiring Agent 演示用模拟数据**，并非字节跳动官方 Wiki、财报或制度原文。  
> 业务名称、组织称谓、财务数字、福利政策等均为 **虚构或公开信息的合理演绎**，仅供招聘对话、JD 生成与候选人问答演示。  
> 请勿将本内容当作真实用工承诺、投资建议或对外发声依据。
