# 【模拟 Wiki】业务与产品矩阵

> 模拟「产品百科」。产品能力描述为演示向整理，细节以实际产品为准。

## 总览

字节跳动的产品可粗分为：

1. **内容消费与创作**（短视频、资讯、直播）
2. **音乐与音频**
3. **企业协作与生产力**
4. **商业化与营销科技**
5. **国际化与其他创新业务**

---

## 1. 抖音（Douyin）

### 产品定位

短视频与直播为主的内容社区，覆盖娱乐、知识、生活、电商、本地服务等场景。

### 核心模块（模拟）

| 模块        | 说明                                 |
| ----------- | ------------------------------------ |
| 推荐 Feed   | 个性化信息流，强算法驱动             |
| 创作工具    | 拍摄、剪辑、特效、模板、草稿         |
| 直播        | 娱乐直播、电商直播、活动直播         |
| 创作者中心  | 数据、收益、粉丝、活动报名           |
| 搜索        | 内容、用户、商品、本地生活等综合搜索 |
| 电商 / 橱窗 | 内容带货与交易闭环（按地区能力差异） |
| 本地生活    | 到店、团购等本地服务（演示口径）     |

### 关键角色

- 用户：刷视频、互动、下单、打赏
- 创作者：生产内容、涨粉、变现
- 商家 / 广告主：获客与转化
- 运营与安全：生态治理、活动、审核策略

### 技术关注点（给研发候选人）

推荐系统、音视频、实时互动、内容安全、大规模存储与推送、端性能。

---

## 2. 今日头条（Toutiao）

### 产品定位

资讯与内容分发平台，强调「你关心的，才是头条」。覆盖图文、小视频、专题等多种形态。

### 典型能力

- 个性化推荐资讯
- 频道与兴趣订阅
- 搜索与热点追踪
- 创作者图文发布与收益

### 与抖音的关系（模拟表述）

同属内容生态，用户场景与内容形态不同；中台能力（推荐、审核、商业化）可能共享，产品策略独立。

---

## 3. 汽水音乐（模拟重点展开）

### 产品定位

面向年轻用户的听歌与音乐社区产品，强调歌单、个性化推荐、社交分享与发现新音乐。

### 核心功能（模拟）

| 功能                | 说明                           |
| ------------------- | ------------------------------ |
| 每日推荐 / 私人雷达 | 基于听歌行为的个性化推荐       |
| 歌单                | 创建、收藏、协作歌单           |
| 歌词与识曲          | 听歌体验增强                   |
| 播客 / 电台（若有） | 长音频场景（按实际版本）       |
| 会员权益            | 音质、下载、广告体验等（演示） |
| 艺人主页            | 作品、动态、歌单               |

### 业务协作方

版权与内容合作、增长投放、商业化、社交传播、品牌活动。

### 候选人卖点话术（模拟）

> 汽水音乐把「听歌」做成可推荐、可社交、可持续运营的内容产品，工程上涉及推荐、音频体验、版权合规与增长实验。

---

## 4. 飞书（Lark / Feishu）

### 产品定位

新一代一站式协作套件：即时通讯、文档、会议、日历、多维表格、审批与开放平台。

### 模块地图（模拟）

- **即时沟通**：单聊、群聊、话题群、机器人
- **云文档**：文档、表格、幻灯片、知识库
- **多维表格**：轻量业务系统搭建
- **会议**：音视频会议、录制、妙记（演示名）
- **日历与待办**：日程协同
- **管理后台**：组织架构、权限、审计
- **开放平台**：机器人、应用市场、API

### 客户类型

互联网、制造业、零售、教育、专业服务等；对安全合规与稳定性要求高。

---

## 5. 商业化业务（模拟）

- 信息流广告、搜索广告、品牌广告
- 营销平台与效果衡量工具
- 电商交易相关商业化能力
- 企业服务销售与续费

## 产品选择建议（给候选人）

- 喜欢消费级增长与推荐：抖音 / 头条方向
- 喜欢音频与内容品味：汽水音乐方向
- 喜欢 ToB 与复杂协作：飞书方向
- 喜欢商业与策略：商业化方向
