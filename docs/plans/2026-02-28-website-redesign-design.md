# 网站重新设计 - 极简主义风格

## 设计概述

这是一个对 Next.js 16 模板项目的全面重新设计，采用现代极简主义设计风格，强调内容优先和优秀的用户体验。

## 设计目标

### 核心原则

- **极简主义**：少即是多，突出内容
- **内容优先**：视觉设计为内容服务
- **响应式**：在所有设备上提供一致的体验
- **可访问性**：确保所有用户都能正常使用
- **性能优化**：快速加载，流畅交互

## 设计方案

### 1. 配色方案

#### 主色调

- **primary**：#3B82F6（蓝色）
- **primary-foreground**：白色

#### 中性色

- **background**：#FFFFFF（白色）
- **foreground**：#111827（深灰）
- **muted**：#F3F4F6（浅灰）
- **muted-foreground**：#6B7280（中灰）
- **accent**：#F9FAFB（极浅灰）
- **accent-foreground**：#374151（灰色）

#### 边框和输入

- **border**：#E5E7EB（浅灰）
- **input**：#D1D5DB（中灰）

### 2. 排版

#### 字体层级

- **H1**：text-4xl sm:text-5xl lg:text-6xl，font-bold，tracking-tight
- **H2**：text-2xl sm:text-3xl lg:text-4xl，font-semibold
- **H3**：text-xl sm:text-2xl，font-medium
- **正文**：text-base sm:text-lg，leading-relaxed
- **辅助文字**：text-sm sm:text-base，text-muted-foreground

#### 间距

- **行距**：1.6 - 1.8（宽松）
- **段落间距**：1.5rem

### 3. 布局结构

#### 网格系统

- 使用 8px 网格系统保持一致性
- 容器宽度：最大 1400px
- 内边距：2rem

#### 页面架构

1. **导航栏**：固定顶部，背景模糊效果
2. **Hero 区域**：大标题、简洁描述、行动按钮
3. **功能特性**：3列卡片布局
4. **快速开始**：代码示例和项目架构
5. **CTA 区域**：底部行动召唤
6. **页脚**：版权信息和链接

### 4. 组件设计

#### 按钮

- 主按钮：蓝色背景，白色文字，圆角，悬停时轻微缩放
- 次按钮：白色背景，灰色边框，圆角
- 按钮高度：48px，padding：16px 32px

#### 卡片

- 白色背景，浅灰色边框，圆角，微妙阴影
- 悬停时轻微上浮（-8px）和阴影增强

#### 输入框

- 浅灰色背景，灰色边框，圆角
- 聚焦时边框变蓝色，添加阴影

### 5. 动画效果

- **淡入淡出动画**：页面加载时的元素淡入
- **卡片悬停**：轻微上浮和阴影变化
- **导航栏滚动**：背景透明度和高度变化
- **按钮悬停**：轻微缩放（1.05倍）
- **平滑滚动**：所有链接点击的平滑滚动效果

### 6. 视觉元素

#### 图标

- 使用简约线条风格的图标
- 大小统一，色彩和谐

#### 分隔线

- 浅灰色细线，用于分隔内容区块

#### 装饰元素

- 微妙的渐变背景
- 卡片边角的轻微圆角
- 内容区块的适度阴影

## 响应式设计

### 移动端

- 导航栏转为汉堡菜单
- 布局变为单列
- 字体大小适当调整
- 按钮宽度变为100%

### 平板端

- 部分布局变为双列
- 字体大小和间距调整
- 导航保持桌面端样式

### 桌面端

- 完整的三列布局
- 最大宽度限制
- 最佳视觉体验

## 技术实现

### 使用的工具

- Next.js 16
- React 19
- TypeScript 5.7
- Tailwind CSS 4
- shadcn/ui 组件库

### 构建命令

- `pnpm install`：安装依赖
- `pnpm dev`：启动开发服务器
- `pnpm build`：构建生产版本
- `pnpm test`：运行测试

## 总结

这个极简主义设计方案提供了一个干净、现代的用户界面，专注于内容展示和优秀的用户体验。通过简化视觉元素和优化交互，我们创造了一个易于使用和维护的网站模板。
